# 🚀 TaskFlow - Full Stack Productivity Management Platform

## 📖 Overview

TaskFlow is a modern full-stack productivity and task management platform designed to help users organize their work, track habits, manage study schedules, and improve productivity through analytics and visual task boards.

The application combines task management, habit tracking, study planning, calendar scheduling, and productivity analytics into a single user-friendly platform.

---

## ✨ Features

### 🔐 User Authentication
- Secure user registration and login
- JWT-based authentication
- Protected routes and APIs
- User-specific data management

### 📋 Task Management
- Create, update, delete, and organize tasks
- Set task priorities
- Add task descriptions
- Track task progress efficiently

### 📌 Kanban Board
- Drag-and-drop task management
- Organize tasks using:
  - 📝 To Do
  - ⚡ In Progress
  - 👀 Review
  - ✅ Completed
- Visual workflow tracking

### 📊 Task Status Tracking
- Monitor task progress in real time
- Update task status dynamically
- Improve productivity through workflow visibility

### 🔥 Habit Tracker
- Create daily habits
- Track consistency and streaks
- Monitor personal growth and productivity habits

### 📚 Study Tracker
- Plan study schedules
- Track completed study sessions
- Monitor learning progress
- Organize subjects and study goals

### 📅 Calendar Integration
- Monthly and weekly task view
- Schedule tasks and deadlines
- Manage events and study plans
- Visual calendar-based planning

### 📈 Analytics Dashboard
- Productivity insights
- Task completion statistics
- Habit consistency reports
- Study progress visualization
- Performance tracking dashboard

### 📱 Responsive User Interface
- Mobile-friendly design
- Tablet and desktop support
- Modern and intuitive user experience

### 🐳 Dockerized Deployment
- Fully containerized application
- Docker Compose support
- Easy local development setup
- Consistent deployment environment

## 🚀 Getting Started

### 📥 Clone Repository

```bash
git clone https://github.com/your-username/taskflow-fullstack-project.git
```

### ⚙️ Backend Setup

```bash
cd backend
mvn clean install
mvn spring-boot:run
```

### 🎨 Frontend Setup

```bash
cd frontend
npm install
npm start
```

### 🐳 Docker Setup

```bash
docker-compose up -d
```
